¿Qué es Jungle Notes?


🎯 Propósito

Plataforma web para centralizar, buscar, descargar y comentar apuntes académicos entre estudiantes universitarios.


⚙️ Stack

  • Laravel 13 + PHP 8.4
  • Livewire v4 + Flux UI v2
  • Tailwind CSS v4
  • SQLite / MySQL
  • Laravel Fortify (Auth)
  • FullCalendar v6

👥 Usuarios

Estudiantes universitarios que necesitan organizar y compartir material de estudio.


✨ Features clave

  • CRUD de Materias, Apuntes, Comentarios, Eventos
  • Bloc de Notas personal
  • Calendario académico
  • API REST
  • Responsive + Dark mode
  • Búsqueda en tiempo real

🗂️ Modelos y Base de Datos


6 modelos con migraciones, claves foráneas y relaciones


ModeloRelacionesCamons clave
UserhasMany Apuntes, Comentarios, Eventos, Notasname, email, password
MateriabelongsTo User, hasMany Apuntesnombre, descripcion, anio
ApuntebelongsTo User/Materia, hasMany Comentariostitulo, descripcion, ruta_archivo
ComentariobelongsTo User/Apuntecontenido
EventobelongsTo Usertitulo, fecha_inicio, fecha_fin, tipo, color
NotabelongsTo Usercontenido, fecha

Relaciones clave: Materia → Apunte (1:N) · Apunte → Comentario (1:N) · User → Todo (1:N)

📝 CRUDs Completos


📚 Materias

CRUD completo con formularios Blade. Grid con cards, contador de apuntes por materia y filtro por año.

📄 Apuntes

CRUD vía Livewire. Subida de archivos (PDF/DOC), búsqueda en tiempo real, tabla relacional con comentarios anidados.

💬 Comentarios

Creación y eliminación inline en cada apunte. Validación de contenido (3-500 chars).

📅 Eventos

CRUD con modal Livewire + FullCalendar. Tipos: examen, presentación, otro. Selector de color.

📝 Notas

Bloc de notas diario con calendario visual. Crear y eliminar notas por fecha.

👤 Usuarios

Registro, login, password reset (Fortify). Perfil con ajustes y foto.

🔌 API REST


MétodoEndpointDescripción
GET/api/apuntesListar todos los apuntes con user y materia
POST/api/apuntesSubir apunte (multipart, PDF/DOC, max 10MB)
GET/api/calendario/eventosEventos del usuario + feriados argentinos

Ejemplo GET /api/apuntes:
[{id, titulo, ruta_archivo, user: {name}, materia: {nombre}}]

🚀 Instalación


# 1. Credenciales Flux
composer config http-basic.composer.fluxui.dev "$USER" "$KEY"

# 2. Dependencias
composer install
npm install && npm run build

# 3. Entorno
cp .env.example .env
php artisan key:generate

# 4. Base de datos
php artisan migrate --seed

# 5. Storage
php artisan storage:link

# 6. ¡A correr!
composer dev
